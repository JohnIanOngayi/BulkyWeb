# BulkyWeb

A website listing for games built on the ASP.NET Core MVC architecture

## Installation

Use the package manager dotnet-sdk

```bash
# after cloning the project
cd src-MVC/
dotnet restore
dotnet build

```

## Start

```bash
# to run project
dotnet run

```
